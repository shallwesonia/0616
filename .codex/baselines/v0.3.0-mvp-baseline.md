# v0.3.0-mvp-baseline

## 版本

`v0.3.0-mvp-baseline`

## 基线内容

- 冻结当前 MVP REST API，并导出 `docs/contracts/openapi.json`。
- 补充 MQTT / AsyncAPI 契约：`docs/contracts/asyncapi.yaml`。
- 建立前端 smoke test 脚本：`npm --prefix frontend run smoke`。
- 新增 Target Registry 标准模型和 API。
- 新增 Robot Config 管理 API。
- 新增 RobotInstance 与 ExecutorInstance 绑定模型。
- 新增 Executor 管理 API。
- 新增规则调度接口和 AgentDecision 消息记录。
- 仿真驾驶舱接入 Target Registry 目标选择、机器人配置新增模式和执行体状态展示。

## 验证要求

- `python -m pytest backend\tests`
- `npm --prefix frontend run build`
- 服务启动后执行 `npm --prefix frontend run smoke`
