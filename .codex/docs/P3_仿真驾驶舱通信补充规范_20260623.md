## 1. 通信边界

P3 不改变机器狗 MQTT 基础契约：

```text
command topic: factory/dogs/{robotCode}/command
result topic:  factory/dogs/{robotCode}/result
```

仿真驾驶舱必须遵守：

- 前端不直连 MQTT Broker。
- 前端不直接调用虚拟机器人执行体。
- 前端不直接修改机器人状态。
- 前端所有动作必须先创建 `Action`。
- `Action` 通过平台消息总成转换为 MQTT command。
- MQTT result 必须先记录为 `MessageRecord`，再标准化为 `Observation`。
- `Observation` 聚合后更新 `CurrentState.stateVersion`。

## 2. 标准链路

```text
/simulation
-> POST /api/v1/actions
-> Action Service
-> MessageRecord(command)
-> MQTT Bridge publish command
-> factory/dogs/{robotCode}/command
-> virtual-robot-runner
-> factory/dogs/{robotCode}/result
-> MQTT Bridge receive result
-> MessageRecord(event)
-> Observation
-> CurrentState
-> TraceSpan
-> WebSocket simulation.snapshot
```

## 4. 消息分类

| 标准分类 | 来源 | 说明 |
|---|---|---|
| Command | Platform API / MQTT Bridge | 平台下发的动作指令 |
| Ack | MQTT result | 指令接收、拒绝、开始处理等确认 |
| Telemetry | MQTT result | 位置、心跳、where 结果等遥测 |
| Event | MQTT result / 平台内部 | 任务开始、任务完成、快照、运行事件 |
| Alert | MQTT result / 异常注入 | 离线、失败、超时、阻塞、消息丢失 |

事件映射：

| event | category |
|---|---|
| `command.accepted` | Ack |
| `command.rejected` | Ack |
| `pose.updated` | Telemetry |
| `where.result` | Telemetry |
| `task.started` | Event |
| `task.succeeded` | Event |
| `task.failed` | Alert |
| `task.timeout` | Alert |
| `device.offline` | Alert |
| `robot.offline` | Alert |
| `action.failed` | Alert |
| `path.blocked` | Alert |
| `interface.timeout` | Alert |
| `message.dropped` | Alert |
| `resource.locked` | Alert |
| `station.unavailable` | Alert |
| `battery.low` | Alert |

## 5. 异常注入规范

异常注入入口：

```text
POST /api/v1/simulation-runs/{runId}/events
```

支持事件：

```text
robot.offline
action.failed
path.blocked
interface.timeout
message.dropped
resource.locked
station.unavailable
battery.low
```

处理规则：

- 异常注入创建 `MessageRecord(event)`。
- 随后创建 `Observation(category=Alert)`。
- `Observation` 更新 `CurrentState.activeEvents`。
- 路径阻塞类事件更新 `CurrentState.resourceStates.blockedPaths`。
- Trace 记录 `observation.{event}` span。
- 默认不向真实机器人网关发布异常控制指令。

## 6. 导出规范

单次 Run 导出入口：

```text
GET /api/v1/simulation-runs/{runId}/export
```

导出内容：

```text
manifest
workspace
run
tasks
actions
observations
currentState
snapshots
traces
mqttMessages
audit
```

导出包不得包含：

- MQTT 密码。
- JWT。
- 证书私钥。
- 数据库连接串。
- Docker secret。
